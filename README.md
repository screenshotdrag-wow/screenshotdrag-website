## 🛠️ Development

### Local Development

Simply open `index.html` in your browser:

```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

### Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel Dashboard
3. Deploy!
