🔒 개인정보처리방침

1. 개인정보의 처리목적

Capture Drag는 다음의 목적을 위하여 개인정보를 처리합니다:

  • Beta Test 참여자 관리 및 Pro Plan 제공
  • 서비스 개선을 위한 피드백 수집
  • 고객 문의 및 불만 처리

2. 개인정보의 처리 및 보유기간

개인정보는 원칙적으로 개인정보의 처리목적이 달성되면 지체없이 파기합니다. 단, 다음의 정보에 대해서는 아래의 이유로 명시한 기간 동안 보존합니다:

  • Beta Test 참여자 정보: Beta Test 종료 후 1년
  • 고객 문의 기록: 3년 (전자상거래법)

3. 처리하는 개인정보의 항목

  • 필수항목: 이메일 주소
  • 선택항목: 이름, 피드백 내용

4. 개인정보의 제3자 제공

Capture Drag는 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다. 다만, 아래의 경우에는 예외로 합니다:

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

5. 개인정보의 안전성 확보조치

개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다:

  • 개인정보 암호화
  • 해킹 등에 대비한 기술적 대책
  • 개인정보에 대한 접근 제한

❓ 자주 묻는 질문 (FAQ)

무료 버전과 프로 버전의 주요 차이점은 무엇인가요?

+

핵심 차이점:
• 📸 캡처 한도: 무료 3개 → 프로 100개 (33배 증가!)
• 🏷️ 워터마크: 무료 버전에는 하단에 워터마크가 있습니다
• 🎨 편집 기능: 두 버전 모두 모든 편집 도구를 사용할 수 있습니다

무료 버전으로도 충분한가요?

+

무료 버전이 충분한 경우:
• 일상적인 스크린샷 캡처
• 가끔씩 편집이 필요한 경우
• 개인 사용 목적
• 앱 기능을 먼저 체험해보고 싶은 경우

언제 프로 버전을 고려해야 하나요?

+

프로 버전이 필요한 경우:
• 업무용 프레젠테이션, 보고서 작성
• 연속적인 캡처가 필요한 대량 작업
• 워터마크 없는 깨끗한 이미지가 필요한 경우
• 3개 캡처로 부족함을 느낄 때

프로 버전은 어떻게 구매하나요?

+

Microsoft Store에서 구매할 수 있습니다:
• 영구 라이선스: $29 (한 번 구매로 평생 사용)

시스템 요구사항은 어떻게 되나요?

+

요구사항:
• OS: Windows 10/11
• .NET 8.0 Runtime
• 메모리: 낮은 사용량 (~200MB)
• CPU: 낮은 사용량 (~3%)

무료 버전에서도 모든 편집 도구를 사용할 수 있나요?

+

네! 무료 버전에서도 모든 편집 기능을 사용할 수 있습니다:
• 펜, 브러시, 지우개 도구
• 화살표, 텍스트, 도형 그리기
• 줌 인/아웃, 팬(이동)
• 실행 취소/다시 실행, 저장 기능
