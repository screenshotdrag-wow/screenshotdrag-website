How to Use?

📸 Single Capture

📹
1

Click Capture (or Alt+S)

2

Capture in progress

3

Capture complete → Thumbnail displayed

4

Drag & drop to target app ✅
※ Ctrl+V paste available

✨ Powerful

📸 Multi Capture

📹
1

Click Multi Capture (or Alt+D)

2

Capture → Free activity → Click capture button on toolbar (or Space bar) → Free activity → Click capture button (or Space bar) → 🔄...

3

ESC → Exit multi capture mode

4

Drag multiple images → Drop all at once ✅
※ Ctrl+V paste available

❓ Frequently Asked Questions (FAQ)

What are the main differences between the Free and Pro versions?

+

Key Differences:
• 📸 Capture Limit: Free 3 → Pro 100 (33x more!)
• 🏷️ Watermark: Free version has a watermark at the bottom
• 🎨 Editing Tools: Both versions include all editing tools

Is the Free version sufficient?

+

The Free version is sufficient when:
• Taking occasional screenshots
• Need editing occasionally
• Personal use
• Want to try the app features first

When should I consider the Pro version?

+

Pro version is needed when:
• Creating business presentations and reports
• Need continuous captures for bulk work
• Need clean images without watermark
• 3 captures are not enough

How do I purchase the Pro version?

+

You can purchase from Microsoft Store:
• Lifetime License: $49 (one-time purchase for lifetime use)

What are the system requirements?

+

Requirements:
• OS: Windows 10/11
• .NET 8.0 Runtime
• Memory: Low usage (~200MB)
• CPU: Low usage (~3%)

Can I use all editing tools in the Free version?

+

Yes! The Free version includes all editing features:
• Pen, brush, eraser tools
• Arrow, text, shape drawing
• Zoom in/out, pan (move)
• Undo/redo, save functionality
