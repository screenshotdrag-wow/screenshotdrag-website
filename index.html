⌨️ Alt+S (단일) / Alt+D (연속) 🖱️ Drag & Drop 🪟 Windows 10/11

📸 가장 빠른 스크린샷 워크플로우

캡처 → 드래그 → 공유
복잡한 단계 없이, 한 번의 드래그로 끝나는 초고속 워크플로우.

⚡

1️⃣ 스마트 캡처

  • Single / Continuous 모드
  • Free: 최대 2장
  • Pro: 무제한 연속 캡처
  • 단축키: Alt + S (단일) / Alt + D (연속)
  • 영역 선택 / 전체화면 / 실시간 썸네일 미리보기
🖱️

2️⃣ 빠르고 직관적인 드래그 앤 드롭

  • 썸네일 바로 드래그 → 원하는 앱에 즉시 드롭
  • 다중 드래그 앤 드롭 지원 (Free: 최대 2개 / Pro: 무제한)
  • 자동 창 숨김: 드래그 순간, 방해 없는 화면
  • 강력한 호환성: Slack, Notion, Discord, Figma 등 주요 앱과 매끄럽게 작동
  • 자동 클립보드 복사: 드롭 후 바로 붙여넣기 가능
💾

3️⃣ 스마트 저장 & 관리

  • 자동 저장: Documents/Screenshots 폴더
  • 파일명 규칙: Screenshot_YYYYMMDD_HHMMSS_GUID.png
  • 용량 자동 관리: 500MB 초과 시 오래된 캡처 자동 삭제
  • 썸네일 갤러리: 최근 캡처 자동 정렬
🧩

4️⃣ 사용자 친화적 UX

  • 트레이 실행 / 자동 시작 (윈도우 부팅 시)
  • 캡처 후 자동 최소화 — 백그라운드에서 깔끔하게 작동
  • 가벼운 애니메이션 — 부드러운 전환, 끊김 없는 경험

📸 Single 캡처

📹 Single 캡처 데모 영상 준비 중

1

Alt+S 누름 (또는 Single 버튼 클릭)

2

마우스로 캡쳐 영역 드래그

3

캡처 완료 → 메인창에 썸네일 표시 (클립보드 자동 복사)

4

썸네일 드래그 앤 드롭 ✅
※ Ctrl+V로 붙여넣기도 가능

🔄 Continuous 캡처

📹 Continuous 캡처 데모 영상 준비 중

1

Alt+D 누름 (또는 Continuous 버튼 클릭)

2

영역 드래그 → 캡처 1, 캡처 2, 캡처 3...

3

ESC, ✕ → 캡쳐 모드 종료

4

Ctrl+클릭으로 드롭 순서 선택 or 마우스로 드래그 → 한꺼번에 드롭 ✅
※ Ctrl+V로 붙여넣기도 가능

🎨 편집 포함 워크플로우

※ 에디터 모드를 항상 활성화 해둘 수도 있습니다. (기본은 해제 상태)

📸 Single 캡처 + 편집

📹 Single 편집 데모 영상 준비 중

1

Alt+S 누름 (또는 Single 버튼 클릭)

2

마우스로 캡쳐 영역 드래그

3

캡처 완료 → 메인창에 썸네일 표시

4

에디터 체크 ✓ → 썸네일 클릭

5

편집창에서 펜/도형/텍스트 편집 → 썸네일에 실시간 반영 ⚡

6

썸네일 잡고 → 드롭 ✅
※ Ctrl+V로 붙여넣기도 가능

🔄 Continuous 캡처 + 편집

📹 Continuous 편집 데모 영상 준비 중

1

Continuous 버튼 클릭 (또는 Alt+D)

2

영역 드래그 → 캡처 1, 캡처 2, 캡처 3...

3

ESC, ✕ → 캡쳐 모드 종료

4

에디터 체크 ✓ → 원하는 썸네일 클릭

5

썸네일 별 개별 편집 → 썸네일 실시간 반영 ⚡

6

Ctrl+클릭으로 드롭 순서 선택 or 마우스로 드래그 → 한꺼번에 드롭 ✅
※ Ctrl+V로 붙여넣기도 가능

🔧 유용한 기능

캡처 후 바로 편집하고 변환하는 스마트한 기능들

✏️

썸네일에서 바로 편집

캡처된 썸네일을 클릭하면 즉시 편집 모드로 전환됩니다.

📹 썸네일 편집 데모 영상 준비 중

  • 펜 도구로 자유롭게 그리기
  • 도형 삽입 (사각형, 원, 화살표)
  • 텍스트 추가 및 편집
  • 실시간 미리보기
🎬

GIF 변환 기능

연속 캡처한 이미지들을 자동으로 GIF로 변환합니다.

📹 GIF 변환 데모 영상 준비 중

  • 연속 캡처 → 자동 GIF 생성
  • 프레임 속도 조절 (0.5s ~ 2s)
  • GIF 품질 설정 (낮음/보통/높음)
  • 파일 크기 최적화
📁

스마트 파일 관리

캡처한 이미지들을 자동으로 정리하고 관리합니다.

📹 파일 관리 데모 영상 준비 중

  • 자동 폴더 생성 (날짜별)
  • 파일명 규칙 적용
  • 용량 관리 (500MB 초과 시 자동 삭제)
  • 최근 캡처 갤러리
⚡

원클릭 공유

편집 완료 후 바로 원하는 앱으로 공유합니다.

📹 원클릭 공유 데모 영상 준비 중

  • 드래그 앤 드롭으로 즉시 공유
  • 클립보드 자동 복사
  • 다중 파일 동시 공유
  • 앱별 최적화된 포맷

🏆 차별화 포인트

기존 방식과의 혁신적 차이

기능 기존 방식 우리 솔루션 개선 효과
캡처 후 처리 캡처 → Ctrl+C → 다른 앱 → Ctrl+V 캡처 → 바로 드래그 앤 드롭 4단계 → 1단계
여러 이미지 개별 캡처 → 개별 처리 연속 캡처 → 다중 드래그 N단계 → 1단계
시간 절약 기준 혁신적 워크플로우 90% 단축
사용 편의성 복잡한 단계 직관적 드래그 앤 드롭 혁신적 UX
