📸
STEP 1

캡처

Ctrl + Shift + S 단축키로 캡처 모드 진입.
마우스로 영역을 선택하세요.

🖼️
STEP 2

확인

Enter 키로 확정하면
하단에 썸네일이 표시됩니다.
(ESC로 취소 가능)

🚀
STEP 3

드래그

썸네일을 드래그해서
원하는 앱에 드롭!
Ctrl+C/V는 필요 없습니다. ✨

🚀 핵심 혁신 기능

기존 스크린샷 도구와는 완전히 다른 워크플로우

⚡

즉시 드래그 앤 드롭

  • 단일 캡처 (Alt+S): 빠른 한 번 캡처
  • 연속 캡처 (Alt+D): 여러 개 연속 캡처
  • 영역 선택: 마우스로 자유 영역 선택
  • 캡처 즉시 드래그: Ctrl+C/V 우회
  • 고해상도: 원본 품질 유지
🔥

연속 캡처 + 다중 드래그

  • 연속 캡처 모드: Alt+D로 활성화
  • 무제한 캡처: 원하는 만큼 계속 캡처
  • 실시간 갤러리: 모든 캡처 즉시 표시
  • 다중 드래그: 여러 이미지 한 번에 처리
  • 생산성 혁신: N단계 → 1단계
🎨

통합 편집 도구

  • 펜 도구: 자유 그리기 및 주석
  • 도형 도구: 사각형, 원, 화살표
  • 텍스트 추가: 설명 텍스트 삽입
  • 색상 선택: 다양한 색상 팔레트
  • 스마트 관리: 썸네일 갤러리

기존 방식과 ScreenshotDrag의 혁신적 차이

기능 기존 방식 ScreenshotDrag 개선 효과

여러 이미지 개별 캡처 → 개별 처리 연속 캡처 → 다중 드래그 N단계 → 1단계
