🔥 Top 10 Most Frequently Asked Questions

The most common questions from Capture Drag users

Q1. How do I take a screenshot? (Basic Usage)

+

🚀 Quick Capture (3 Steps):

1️⃣ Press Alt + S
2️⃣ Drag mouse over desired area
3️⃣ Press Enter to confirm

📸 Other Capture Modes:
• Full Screen: Select area mode → Click full screen
• Window Capture: Select window mode → Hover over window → Click
• Continuous Capture: Alt + D → Capture multiple times

💡 Key Tips:
• ESC key: Cancel anytime
• Drag & Drop: Drag thumbnail directly to other apps!

Q2. Hotkeys (Alt+S, Alt+D) are not working!

+

🔍 Checklist:

1. Check if program is running
• Look for tray icon (📷 icon)
• If not found, launch the program

2. Check for conflicts with other programs
• Windows Snipping Tool may use same hotkey
• Close conflicting programs and retry

3. Administrator permission issue
• If target app has admin rights, run Capture Drag as admin
• Right-click program → "Run as administrator"

4. Restart program
• Right-click tray icon → Exit → Relaunch

✅ Normal Operation Check:
• Alt+S → Full screen overlay appears
• Alt+D → Overlay + toolbar appears

Q4. How do I paste captured images into other apps?

+

🎯 Drag & Drop Method:

1. Thumbnail displayed at bottom of main window
2. Drag thumbnail with mouse
3. Drop onto desired app window
4. Image automatically inserted!

✅ Supported Apps:
• Messengers: Telegram, KakaoTalk, Discord, Slack
• Browsers: Chrome, Edge, Firefox
• Office: PowerPoint, Word, Excel
• File System: File Explorer, Desktop, OneDrive
• Design: Photoshop, Figma, Notion

💡 Additional Features:
• Auto clipboard copy: First image automatically copied to clipboard
• Multiple drag: Select multiple thumbnails and drag at once

Q5. Where are captured images saved?

+

💾 Default Save Location:
C:\Users\[Username]\Pictures\ScreenshotDrag\

📁 File Name Format:
Screenshot_YYYYMMDD_HHMMSS.png
Example: Screenshot_20250121_153042.png

⚙️ Change Save Settings:
1. Open Settings window
2. Click "Browse" in "Save Folder" section
3. Select desired folder

💡 Tips:
• Auto-save: Enabled by default
• Manual save: Can disable "Auto-save" in Settings
• After program exit: Saved files remain

Q6. How do I edit images? (Text, Pen, Blur, etc.)

+

🎨 How to Use:
1. Select thumbnail → Click "Edit" button
2. Edit window opens
3. Select tool from left toolbar

🛠️ Available Tools:

1. Add Text
• Select text tool → Click canvas → Type
• Double-click to edit
• Adjust font size, color, weight

2. Pen/Marker Drawing
• Select pen tool → Draw freely with mouse
• Adjustable thickness
• Multiple colors supported

3. Blur/Mosaic
• Select blur or mosaic tool
• Adjust intensity
• Drag to select area → Auto apply

4. Draw Shapes
• Add rectangles, circles, etc.
• Adjust color and thickness

⌨️ Edit Shortcuts:
• Ctrl + Z: Undo
• Ctrl + Y: Redo
• Delete: Delete selected element

Q7. How do I take multiple screenshots continuously?

+

🚀 Multi-Capture Usage:
1. Press Alt + D
2. Capture first area → Press Enter
3. Click "Next Capture" button (or Alt+D again)
4. Capture second area → Press Enter
5. Continue repeating...
6. Click "End" button to finish

📸 Multi-Capture Mode Features:
• Mode switching: Switch area/full/window modes in toolbar
• Continuous work: Capture quickly without gaps
• Toolbar display: Small toolbar floats at top for convenience

💡 Use Cases:
• Presentation prep: Capture multiple slides
• Tutorial creation: Step-by-step screenshots
• Document work: Quickly capture multiple sections

Q8. Does the program slow down or use too much memory?

+

📊 Memory Usage:

Captures Memory Status
0-3 ~80-120 MB ✅ Very light
50 ~400 MB ✅ Good
100 ~800 MB ⚠️ Normal

⚡ Performance Tips:
1. Delete old captures
2. Keep recommended limit (Free: 3, Pro: 50-70)
3. Restart program after long use

✅ Background Load:
• Idle: CPU 0-1%
• Capturing: CPU 5-15%
• Editing: CPU 10-30%

Q9. Can I change the hotkeys to different keys?

+

🔄 Current Hotkeys:
• Alt + S: Single capture
• Alt + D: Multi-capture

💡 If you want different shortcuts:
1. Use Windows shortcut change tool (external program)
2. Send feedback: Custom hotkeys planned for future update

⚠️ Conflict Prevention:
• Selected combination doesn't conflict with default Windows hotkeys
• If conflicts with other programs: Recommend changing that program's hotkey

Q10. Do captures disappear when I close the program?

+

✅ Auto-save Enabled (Default):
• All captures saved as files
• Retained after program exit
• Can find in Pictures\ScreenshotDrag folder

⚠️ Auto-save Disabled:
• Stored in memory only
• Disappears on program exit
• Important captures need manual save

🔄 Program State Behavior:

Minimize to Tray:
• Capture history retained in memory
• Can restore anytime

Complete Exit:
• Memory captures deleted
• Saved files retained
• Next launch starts with empty state

💡 Recommended Setting:
✅ Keep auto-save enabled (default)
→ Safely preserve all captures
→ No worries even after program exit
